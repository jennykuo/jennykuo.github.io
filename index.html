What I Do

PROCESS

1. Define

Visualize. Outline the strategic opportunity. Identify problems and solutions. Ask questions. Brainstorm. Organize thoughts.

2. Research

Understand the problem, empathize with the person. Explore the competitive landscape. Define design goals and user requirements. Set target metrics.

3.Collaborate

Empower different stakeholders. Bring them into the process. Eliminate inefficiences. Get input and work cross-functionally. Harness synergies and work in alignment.

4. Design

Combine effective interfaces, smart interactions, and consistent visual language into a captivating design that resonates with the target audience.

5. Build

Map design to code. Smooth constraints. Solve edge cases. Move that button 3 pixels tto tthe left. Fill the gap between functional and delightful.

6. Refine

Test prototypes with real people against actual problems. Combine insights with intuition and experience. Evaluate against target metrics. Iterate and improve in pursuit of the ultimate user experience.
